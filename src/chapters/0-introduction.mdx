# Chapter 0: Introduction

This project documents my journey building a small physics engine and simulation framework in JavaScript. It follows the structure of The Nature of Code by Daniel Shiffman — a book that explores computational physics through interactive sketches — while going deeper into the theory wherever a topic genuinely warrants it.

The long-term goal is to eventually write a rendering and physics engine in C++. This is the first step: working through the foundational concepts in a high-level environment before descending to lower-level systems. JavaScript lets me iterate quickly, visualize results immediately, and focus on understanding before optimizing.

Rather than implementing every example from the book, I concentrate on the core sketch of each chapter alongside a concise explanation of the ideas behind it. When a topic is especially interesting — or underexplained in the source material — I dig into the theory and implement my own version from scratch. Perlin noise is one such example.
