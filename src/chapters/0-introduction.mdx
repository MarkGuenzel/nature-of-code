# Chapter 0: Introduction

This website documents my journey into building a small physics engine and simulation framework. As a starting point, I am primarily following The Nature of Code by Daniel Shiffman, which explores computational physics concepts using JavaScript and the p5.js library. I use the same technologies to implement interactive sketches and experiments.

One of my long-term goals is to eventually develop my own rendering and physics engine in C++. This project represents the first step toward the goal, allowing me to explore the underlying concepts in a more acessible enviroment before moving to lower-level systems.

The structure of this website loosely follows the chapters  of The Nature of Code. Rather than implementing every example from the book, I focus on the main sketch of each chapter and provide a concise summary of the key concepts involved. When the book introduces ideas that are especially interesting or not covered in depth - such as Perlin Noise - I dive deeper into the theory and attempt to implement my own versions from scratch.
